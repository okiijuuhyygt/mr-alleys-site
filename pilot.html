AI Agent 導入指南

一頁看懂:要準備什麼、要知道什麼、能拿它做什麼。看到想要的,直接打勾 ✓

① 要準備的東西

硬體

  • Mac mini(或任一台常開的電腦)Agent 的家。雲端或本地都行,重點是 24 小時待機,它才能隨時幫你做事
  • 大螢幕看 Agent 工作、對長文件時舒服很多
  • 指紋辨識很重要——授權、帳號密碼確認都靠它,安全又快

軟體

  • Claude CodeAgent 本體
  • Discord + 手機跟 Agent 對話的窗口,頻道分類=分類你的生活與工作
  • Obsidian資料與記憶——給「人」看的筆記庫
  • GitHub資料與記憶——給「AI」看的版本庫,兩邊互為備份
  • Wispr Flow / Typeless語音輸入,用講的就能指揮 Agent

② 玩 Agent 前要知道的事

  • 死機怎麼辦額度用完 → 切換低階模型或等重置時間;伺服器掛掉、對話太滿 → 重啟就好
  • 備份習慣每 4hr / 6hr / 12hr 自動備份+每日收官;開始玩之前,先把既有資料、之前的對話備份起來
  • 記憶(連貫性)Agent 每次醒來都是新的,靠檔案接續記憶——記憶設計是靈魂
  • 切換模型貴的模型做難的事,便宜的做日常,學會切換省很多
  • 權限(bypass permission)放行它才跑得順,但要懂自己放了什麼
  • 搬家自由Anthropic(Claude Code)↔ OpenAI(Codex)可以互搬,記憶放檔案就不怕被綁死
  • Skill 與串接 API把常做的事寫成技能,接上各種服務,越用越強
