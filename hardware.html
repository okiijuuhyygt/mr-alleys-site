軟體與帳號

下面這些安裝當天再一起弄就好,先不用急著辦。列出來只是讓你心裡有底。
Apple ID 共用
用現有的 Apple ID 即可、共用一個。因為 Agent 的資料庫(Obsidian)會放在 iCloud 裡,共用同一個 Apple ID,手機才能跟主機完整整合、隨時看到同一份資料。
iCloud 儲存空間
安裝當天再訂閱 · 建議 NT$100/月(200GB) 方案
放資料庫用,200GB 很夠。當天開通就好。
需要先辦好的帳號
Discord · GitHub · Obsidian
這三個註冊一下(免費),安裝當天會用到。Discord=之後跟 Agent 對話的窗口;GitHub=AI 看的備份庫;Obsidian=人看的筆記庫。
Google 帳號
延用你現有的就好。日曆的部分安裝時再確認你習慣用 Google 還是 Apple 行事曆。
暫時不需要 跳過
Tailscale · ChatGPT
這兩個目前先不用辦,之後有需要再說。
⚠️ 開工前務必做一件事:把現有的工作檔案完整備份一份(外接硬碟或雲端都可以)。這是鐵則——Agent 有獨立工作區、跟你原本的檔案分開,但先備份最安全。
